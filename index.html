Tablib: Pythonic Tabular Data¶

Tablib is an MIT Lisenced format-agnostic tabular dataset library, written in Python. It allows you to import, export, and manipulate tabular data sets. Advanced features include, segregation, dynamic columns, tags & filtering, and seamless format import & exmport.

I recommend you start with Installation.
